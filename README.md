# Setup

### Prerequisites

- Node.js (https://nodejs.org)
- npm (comes with Node.js)

### Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
